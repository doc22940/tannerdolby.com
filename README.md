# Local Development

### 1. Fork this repo

### 2. Clone your own local copy of the project

```
git clone git@github.com:tannerdolby/tannerdolby.com.git
```

### 3. Install dependencies

```
npm install
```

### 4. Build 

```
npm run build
```

### 5. Serve locally

```
npm run serve
```
